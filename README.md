# Restaurant_terminal

Данный проект преднезначен для сотрудников ресторана - Пиццерия.

Restaurant_terminal позволяет выполнять необходимые операции для работы:
* представление меню ресторана
* работа с заказам  в режиме кассира
* работа со счетом
* оплата счета
